# 🛒 ZynKart - E-Commerce Website

ZynKart is a modern and responsive e-commerce web application built using **React.js** and **Tailwind CSS**.
The project provides a smooth shopping experience with dynamic product handling, routing, cart management, and responsive UI design.

---

## 🚀 Live Demo

https://zynkart.netlify.app/

---

## 👨‍💻 About Project

ZynKart is a modern and responsive e-commerce web application built using React.js and Tailwind CSS.
The project includes user authentication with Clerk, location fetching functionality, dynamic routing, cart management using Context API, and a clean responsive UI to provide a real-world shopping experience.

---

## 🎯 Purpose

This project was created for self-improvement and to gain practical experience in building real-world React applications using modern frontend technologies and state management concepts.

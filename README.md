#  E-Commerce Website (React)

This is a fully functional E-Commerce web application built using React and Tailwind CSS.  
Users can browse products, add items to cart, delete items to cart, increament and decreament quantity, sign in and sign out, filter price and product, choose category and brand.  
The UI is clean, responsive, and optimized for user experience.

##  Features

-  Fully Responsive UI

-  Dynamic Cart Price Calculation
-  User Authentication (Login / Signup)
-  Checkout Page
-  Order Summary
